# Mini Compiler

Mini Compiler is a compiler design project developed to demonstrate the fundamental phases involved in translating source code into machine-understandable instructions. The system processes code written in a simplified programming language and performs various compilation stages, including lexical analysis, syntax analysis, semantic analysis, and code generation.

The project provides a practical understanding of compiler construction concepts and illustrates how programming languages are internally processed and executed.

---

## 🚀 Features

* Lexical analysis for token generation
* Syntax analysis and parsing
* Semantic error checking and validation
* Intermediate code generation
* Error detection and handling
* Optional code optimisation techniques
* Target code generation
* User-friendly interface for code input and output

---

## ⚙️ Compiler Phases

### 1. Lexical Analysis

* Scans source code character by character
* Identifies tokens such as:

  * Keywords
  * Identifiers
  * Operators
  * Constants
  * Special symbols

### 2. Syntax Analysis

* Validates grammatical correctness of the source code
* Generates parse trees or syntax structures
* Detects syntax errors

### 3. Semantic Analysis

* Performs type checking
* Verifies variable declarations
* Validates scope rules
* Detects semantic errors

### 4. Intermediate Code Generation

* Converts source code into an intermediate representation
* Simplifies further optimisation and code generation

### 5. Error Handling

* Detects lexical, syntax, and semantic errors
* Provides meaningful error messages

### 6. Code Optimisation (Optional)

* Eliminates redundant instructions
* Improves execution efficiency

### 7. Target Code Generation

* Produces executable or machine-understandable code
