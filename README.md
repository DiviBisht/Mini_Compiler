# 🎵 Emotion Music Detector

Emotion Music Detector is an AI-powered application that detects a user's emotional state through facial expressions and recommends music based on the detected mood. By combining Computer Vision, Machine Learning, Emotion Recognition, and Music Recommendation techniques, the system delivers a personalised and interactive music experience.

The application captures facial expressions using a webcam or uploaded image, analyses the detected emotions, and suggests songs that best match the user's current emotional state.

---

## 🚀 Features

* Real-time emotion detection using facial expressions
* Recognition of multiple emotions:

  * 😊 Happy
  * 😢 Sad
  * 😠 Angry
  * 😐 Neutral
  * 😲 Surprised
  * 😨 Fearful
* Face detection and facial feature analysis using computer vision
* Mood-based music recommendation system
* Instant emotion prediction and song suggestions
* Interactive and user-friendly interface
* Optional emotion-related image display
* Personalised music recommendations

---

## ⚙️ How It Works

1. The user uploads an image or enables the webcam.
2. The system detects the face using computer vision techniques.
3. Facial features are extracted and processed.
4. The trained machine learning model predicts the user's emotion.
5. The detected emotion is displayed.
6. Music recommendations are generated according to the identified mood.
